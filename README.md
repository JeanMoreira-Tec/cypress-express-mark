## Testes automatizados com Cypress

_Pare reproduzir os testes é necessário fazer o deploy da aplicações web de gerenciamento de terefas localmente._

**Acesse a pasta API e Web execute os comando abaixo:**  

### Comandos com yarn:

* yarn install
* yarn db:init
* yarn dev

### Comandos com npm:

* npm install
* npm run db:init
* npm run dev
